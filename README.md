# RickRoll
Bring Rick Astley himself into your world

# Commands
• `/rickroll <player>` - Start rickrolling a specific player

# Configuration

### • Config.yml
```yaml
# distance between player and rick astley
npc-distance: 3.0
# swinging speed
swing-speed: 0.4
```
